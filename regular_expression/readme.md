# 正则表达式

正则表达式（regular expression）,即规律表达式，通过描述文本出现的规律来对文本进行查找/匹配/替换等操作。正则表达式作为一种语言，因其强大的文本处理功能，在近乎所有软件中都得到了支持。正则表达式是非常值得学习的。
